# Rubicon
Issue tracking and community feedback for Rubicon — **Automated trading platform built for serious traders.**

Rubicon is a desktop application that brings systematic, safety-first automated trading to futures, crypto, and an expanding range of markets. Built by traders, for traders — with the discipline of a prop firm and the flexibility of a personal trading desk.

This repository is for **issue tracking, bug reports, feature requests, and community feedback** only. The Rubicon source code is proprietary and not hosted here.

---

## What is Rubicon?

A trader-built, AI-augmented automated trading platform that combines premium execution, real-time analytics, and operator-grade safety controls in one desktop application.

Rubicon enforces safety at the engine level — drawdown caps, daily loss limits, EOD flatten, consecutive-loss circuit breakers, mandatory brackets — so the bot cannot violate the rules even if a strategy goes haywire. Whether you're trading a prop firm evaluation, managing your own capital, or running multi-account portfolios, Rubicon scales with your needs.

**Supported markets:**
- Futures (CME, CBOT, NYMEX, COMEX)
- Crypto (major exchanges via REST and WebSocket)
- More platforms and asset classes on the roadmap

**Supported brokers and integrations:**
- Rithmic (multi-firm gateway)
- Topstep, MFFU, TPT, Tradeify, FundedNext (prop firm evaluations)
- Tradovate
- NinjaTrader 8 (ATI bridge)
- Major crypto exchanges
- TradingView webhook publishing

**Key features:**
- Bring-your-own-key LLM integration (Hermes AI assistant)
- Real-time performance dashboard with R-multiple analytics, time-of-day heatmaps, setup attribution
- Multi-account support
- Customizable strategy framework with safety constraints
- Audit log for full trade history
- Premium UX built to professional desktop platform standards

**Pricing tiers** (subject to change before launch):
- Solo — $79/mo
- Pro — $179/mo
- Elite — $349/mo
